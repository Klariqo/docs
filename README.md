# Klariqo docs

Public documentation for Klariqo, the compliance layer for call centers on the dialer you already run. Live at docs.klariqo.com.

Built with Mintlify. Pages are MDX files in this repo, and `docs.json` holds the navigation and configuration.

## Local preview

Install the Mintlify CLI and run the dev server from the repo root, where `docs.json` lives:

```
npm i -g mint
mint dev
```

Preview at `http://localhost:3000`.

## Contributing

Read `AGENTS.md` before editing. The non-negotiables: describe evidence and provenance, never legal compliance; no internal infrastructure or implementation detail; synthetic examples only; no em dashes.
